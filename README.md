# CarouselSwiftUI

This is a simple SwiftUI project demonstrating how to build a smooth and visually appealing carousel view.  
The goal of this project is to understand **scroll alignment**, **animation**, and **content scaling** while scrolling.

- Built entirely using **SwiftUI**.
- Horizontal carousel with center alignment.
- Smooth scaling effect while scrolling.
- Page indicator dots for the active card.
- Uses `scrollPosition` and `scrollTargetBehavior(.viewAligned)` (iOS 17+).
